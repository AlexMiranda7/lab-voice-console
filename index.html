How to configure MAKE
1
Create a Custom Webhook trigger in MAKE
2
Add a Parse JSON module → JSON string: 1.data
3
Add Slack → Create Message → Channel: 2.channel, Text: 2.formatted_message
Sample payload 
{
  "brand_name": "New Relic",
  "channel": "lab-newrelic",
  "input": "Can you send the report?",
  "translation": "¿Puedes enviar el informe?",
  "intention": "El cliente necesita el informe",
  "response": "Here's the report...",
  "formatted_message": "*New Relic* — Client message\n> Can you send the report?"
}
⚡ Note: The app calls /.netlify/functions/slack-proxy which forwards to MAKE. No CORS issues.
